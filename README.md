# kindle-highlight-to-scrapbox

Convert Kindle highlights to Scrapbox.

## Build bookmarklet

1. Edit `src/bookmarklet.js` and set your Scrapbox user name.
2. Run build script.

   ```bash
   $ npm run build
   ```

3. Use `dist/bookmarklet.js` as bookmarklet.

## Usage

1. Open Kindle highlight page.
2. Run bookmarklet.
